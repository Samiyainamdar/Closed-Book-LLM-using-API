# Closed-Book LLM using API

##  Project Overview
This project is a Flask-based API that simulates a closed-book Large Language Model (LLM).  
It answers questions strictly based on the provided dataset without external knowledge.

##  Features
- REST API built using Flask
- Accepts POST requests
- Processes structured data (CSV, JSON, TXT)
- Returns model-generated responses
- Designed for controlled, dataset-based Q&A

## Tech Stack
- Python
- Flask
- Pandas
- Git & GitHub

## How to Run

1. Activate virtual environment
2. Install dependencies:
   pip install -r requirements.txt
3. Run the app:
   python app.py
4. Test using Postman with POST request
